# **App Name**: Retail Insights Dashboard

## Core Features:

- Sales Overview: Display key sales metrics such as total revenue, sales growth, and top-selling products.
- Market Basket Analysis: Identify frequently co-occurring products to optimize product placement and promotions.
- Customer Segmentation: Segment customers based on purchase history, demographics, and behavior.
- Data Visualization: Enable interactive charts and graphs for easy exploration of sales, customer, and product data.

## Style Guidelines:

- Primary color: Deep Blue (#3F51B5) for a professional and trustworthy feel, reflecting stability and data focus.
- Background color: Light Gray (#F5F5F5) for a clean, modern backdrop, ensuring data visibility.
- Accent color: Teal (#009688) to highlight key metrics and interactive elements, providing visual distinction.
- Body and headline font: 'Inter', a grotesque-style sans-serif font known for a modern and neutral aesthetic, ideal for displaying data.
- Use consistent, minimalistic icons to represent data categories and actions. Ensure icons are intuitive and align with the overall design language.
- Implement a clean and structured layout with clear hierarchy, emphasizing the most important metrics. Utilize a grid system to maintain consistency.
- Incorporate subtle transitions and animations for interactive elements such as chart updates and data filtering, enhancing user engagement.
